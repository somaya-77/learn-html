    HTML Tutorial

    It is for Hyper Text Markup Language, the standard markup language for creating Web pages, describes the structure of a Web page & consists of a series of elements.

    Elements

    Elements are a component of HTML that contains opening tag and closing tag. used to denote document parts such as headers, paragraphs and footers.

    View
    Attributes

    Attributes are an HTML component that sets the behavior of an element or its layout in other languages that contains name and value.

    View

    HTML Element Reference:

    Basic HTML

    <!-- -->

    Defines a comment.

    <!DOCTYPE>

    Defines the document type.

    <html>

    Defines an HTML document.

    <head>

    Contains metadata/information for the document.

    <title>

    Defines a title for the document.

    <body>

    Defines the document's body.

    <h1>to<h6>

    Defines HTML headings.

    <p>

    Defines a paragraph.

    <br>

    Inserts a single line break.

    <hr>

    Defines a thematic change in the content.
    Links / Lists
    <a>
    Defines a hyperlink.
    معرف الارتباط التشعبي
    <link>
    Defines the relationship between a document and an external resource.
    <nav>
    Defines navigation links.
    لتحديد روابط التنقل
    <ul>
    Defines an unordered list.
    لتحديد قوائم غير مرتبة
    <ol>
    Defines an ordered list.
    لتحديد قوائم مرتبة
    <li>
    Defines a list item.
    لتحديد عنصر القائمة
    <dl>
    Defines a description list.
    لتحديد قائمة وصفية
    <dt>
    Defines a term/name in a description list.
    اسم القائمة الوصفية
    <dd>
    Defines a description of a term/name in a description list.
    وصف القائمة
    Images
    <img>
    Defines an image.
    معرف الصور
    <map>
    Defines a client-side image map.
    تحديد خريطة الصورة للعميل
    <area>
    Defines an area inside an image map.
    لتحديد منطقة داخل خريطة الصورة
    <canvas>
    Used to drow graphics scripting (usually JS).
    تستخدم للرسومات عبر البرمجة النصية
    <figcaption>
    Defines a caption for a <figure> element.
    لتوضيح تسمية العنصر
    <figure>
    Specifies self-contained content.
    تحديد محتوي قائم بذاتة
    <picture>
    Defines a container for multiple image resources.
    تحديد الصور المتعددة
    <svg>
    Defines a container for SVG graphics.
    خاص بتحديد الرسومات
    Audio / Video
    <iframe>
    Defines an online frame.
    اضافة فيديو اونلاين
    <audio>
    Defines sound content.
    اضافة مقطع صوتي
    <source>
    Defines multiple media resources for media elements.
    تحديد موارد الوسائط
    <video>
    Defines a video or movie.
    اضافة مقطق فيديو
    <track>
    Defines text tracks for media elements.
    تحديد مسارات النص لعناصر الوسائط
    Forms and Input
    <form>
    Defines an HTML form for user input.
    لإدخال المستخدم البيانات المطلوبة
    <input>
    Defines an input control.
    يحدد عنصر تحكم الإدخال
    <textarea>
    Defines a multiline input control.
    منطقة ادخال متعدد الاسطر
    <button>
    Defines a clickable button.
    الزر القابل للنقر من قبل المستخدم
    <select>
    Defines a drop-down list.
    قائمة منسدلة
    <optgroup>
    Defines a group of releted options in a drop-down list.
    مجموعة قوائم منسدلة
    <option>
    Defines an option in a drop-down list.
    يحدد خيارا في قائمة منسدلة
    <lable>
    Defines a lable for an <input> element.
    يحدد تسمية المدخل
    <fieldset>
    Groups related elements in a form.
    المجموعات المرتبطة بعناصر النموذج
    <output>
    Defines the result of a calculation.
    Tables
    <table>
    Defines a table.
    محدد الجدول
    <caption>
    Defines a table caption.
    عنوان للجدول
    <th>
    Defines a header cell in a table.
    لتحديد الخلية الراسية للجدول
    <tr>
    Defines a row in a table.
    لتحديد صف للجدول
    <td>
    Defines a cell in a table.
    لتحديد خلية للجدول
    <thead>
    Groups the header content in a table.
    لتجميع محتوي عناوين الجدول
    <tbody>
    Groups the body content in a table.
    لتجميع المحتوي الاساسي للجدول
    <tfoot>
    Groups the footer content in a table.
    لتحديد اطراف الجدول
    <col>
    Specifies column properties for each column within a <colgroup> element
    يحدد خصائص العمود لكل عمود داخل ملف
    <colgroup>
    Specifies a group of one or more columns in a table for formatting.
    لتحديد عامود
    Formatting
    <abbr>
    Defines an abbreviation or an acronym.
    يحدد اختصارا
    <address>
    Defines contact information for the author/owner of a document/article.
    يحدد معلومات الاتصال لمالك او مؤلف المستند
    <b>
    Defines bold text.
    لجعل النص سميك
    <bdi>
    Isolates a part of text that might be formatted in a different direction from other text outside it.
    يعزل جزءًا من النص قد يتم تنسيقه في اتجاه مختلف عن النص الآخر
    <bdo>
    Overrides the current text direction.
    يتجاوز اتجاه النص الحالي
    <blockquote>
    Defines a section that is quoted from another source.
    يحدد قسمًا مقتبسًا من مصدر آخر
    <cite>
    Defines the title of a work.
    يقوم بتحديد عنوان العمل
    <code>
    Defines a piece of computer code.
    لجعل النص كنص الاكواد
    <del>
    Defines text that has been deleted from a document.
    يحدد النص الذي تم حذفه
    <em>
    Defines emphasized text .
    لجعل النص مائل
    <i>
    Defines a part of text in an alternate voice or mood .
    يحدد جزءًا من النص بصوت
    <mark>
    Defines marked/highlighted text .
    لتمييز النص بالون الاصفر
    <meter>
    Defines a scalar measurement within a known range (a gauge) .
    يحدد قياسًا ضمن نطاق معروف (مقياس)
    <pre>
    Defines preformatted text .
    يقوم بحساب المسافات بين النصوص
    <small>
    Defines smaller text .
    لتصغير حجم النص
    <sub>
    Defines subscripted text .
    لجعل النص اسفل من باقي النصوص
    <sup>
    Defines superscripted text .
    لجعل النص اعلي من باقي النصوص
