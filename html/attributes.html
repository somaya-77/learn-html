The Attributes

class :

Global Elements.

Specifies one or more classnames for an element.

لتحديد اسم الفئة لعنصر او اكثر.

id :

Global Attributes

Specifies a unique id for an element.

لتحديد معرف العنصر الذي لا يتم تكراره.

lang :

Global Attributes

Specifies the language of the element is content.

تحديد لغة المحتوي.

hreflang :

<a> , <link> , <area> , <base>

Specifies the language of the linked document.

لتحديد لغة المستند المرتبط.

Accept :

<input>

Specifies the types of files that the server accepts(Only for type="file").

لتحديد انواع الملفات التي يقبلها الخادم لانواع الملفات فقط.

accept-charset :

<form>

Specifies the character encodings that are to be used for the form submission.

لتحديد التراميز التي يتم استخدامها لارسال النموذج.

action :

<form>

Specifies where to send the form-data when a form is submitted.

يحديد مكان ارسال بيانات النموذج.

alt :

<area> , <img> , <inout>

Specifies an alternate text when the original element file.

لتحديد نص بديل.

width :

<canvas> , <progress> , <param> , <embed> , <iframe> , <img> , <input>

Specifies the width of the element.

تحديد عرض العنصر.

src :

<audio> , <video> , <embed> , <iframe> , <img> , <input> , <script> , <source> , <track>

Specifies the URL of the media file.

تحديد مسار العنوان.

async :

<script>

Specifies that the script is executed asynchornously (only for external scripts).

لتحديد ان البرنامج النصي يتم تنفيذة بشكل غير متزامن للنصوص الخارجية فقط.

autocomplate :

<form> , <input>

Specifies whether the <form> or the <input> element should have autocomplate enabled.

لتفعيل اكمال النموذج تلقائيا.

autoplay :

<audio> , <video>

Specifies that the audio / video will start playing as soon as it is ready.

لتشغيل الصوت الو الفيديو تلقائيا.

autofocus :

<buttom> , <input> , <select> , <textarea>

Specifies that the element should automaticlly get foucs whene the page loads.

للتركيز علي العنصر عند تحميل الصفحة.

charset :

<meta> , <script>

Specifies the character encoding.

لتحديد ترميز العنصر.

cheked :

<input>

Specifies that an <input> element should be pre-selected when the page loads (for type="checkbox" or type="radio").

لتحديد العنصر مسبقا.

cite :

<blockquote> , <del> , <q> , <ins>

Specifies a URL which explains the quote / inserted text.

لتحديد اللينك علي حسب نوعه.

cols :

<textarea>

Specifies the visible width of a text area.

لتحديد مساحة مدخل.

colespan :

<td> , <th>

Specifies the number of columns a table cell should span.

لتحديد عدة اعمدة.

content :

<meta>

Gives the value associated with the http-equiv or name attribute.

ليعطي قيمة مرتبطة او سمة الاسم.

contenteditable :

Global Attributes

Specifies whether the content of an element is editable or not.

تحديد السماح لتحرير المتحوي او منعة.

controls :

<audio> , <video>

Specifies that audio/video controls should be displayed.

لاظهار عناصر التحكم.

coords :

<area>

Specifies the coordinates of the area.

لتحديد احداثيات المنطقه.

data :

<object>

Specifies the URL of the resource to be used by the object.

لتحديد مسار او لينك الكائن.

data-* :

Global Attributes

Used to store custom data private to the page or application.

يتم استخدامة لتخزين المخصصة بالصفحة او التطبيق.

datatime :

<del> , <ins> , <time>

Specifies the data and time.

لتحديد التاريخ و الوقت.

defer :

<script>

Specifies that the script is executed when the page has finished parsing (only for external scripts).

لتفيذ البرنامج النصي عند انتهاء الصفحة من التحليل.

default :

<track>

Specifies that the track is to be enabled if the user's prefernces do not indicate that another track would be more appropriate.

لوضع القيمة الافتراضية اذا لم يحدد مسار ملائم.

dir :

<Global Attributes>

Specifies the text direction for the content in an element.

لتحديد اتجاة النص للعنصر.

dirname :

<input> , <textarea>

Specifies that the text direction will be submitted.

لتحديد اتجاه النص الذي سيتم تقديمة.

disabled :

<button> , <fieldset> , <input> , <optgroup> , <option> , <select> , <textarea>

Specifies that the Specified element / group of elements should be disables.

لتعطيل العنصر او مجموعة عناصر.

download :

<a> , <area>

Specifies that the target will be downloaded when a user clicks on the hyperlink

لتحميل المحتوي او الهدف عند النقر علي الارتباط التشعبي.

draggable :

Global Attributes

Specifies whether an element is draggable or not.

لالغاء او سماح سحب العنصر.

enctype :

<form>

Specifies how the form-data should be encoded when submitting it to the server (only for method="post").

لتحديد طريقه تشفير البيانات عند ارسالها.

for :

<lable> , <output>

Specifies which form element(s) a lable / calculation is boun to.

لارتباط عناصر النموذج.

formaction :

<button> , <input>

Specifies where to send the form-data when a form is submitted. Only for type="submit".

لتحديد مكان الارسال للنوع ارسال فقط.

hidden :

Global Attributes

Specified that an element is not yet, or is no longer, relevant.

لاظهار او اخفاء العنصر.

high :

<meter>

Specifies the range that is considered to be a high value.

لتحديد قيمة الارتفاع.

http-equiv :

<meta>

Provides an HTTP header for the information / value of the content attribute.

لتوفير المعلومات الخاصة بسمة المحتوي.

ismap :

<img>

Specifies an image as a server-side image map.

تحديد صورة كخريطة.

lable :

<track> , <option> , <optgroup>

Specifies the title of the text track.

تحديد عنوان مسار النص.

list :

<input>

Refers to a <datalist> element that contains pre-defined options for an <input> element.

لاضافة عده خيارات.

loop :

<audio> , <video>

Specifies that the audio / video will start over again, every time it is finished.

لتكرار التشغيل دون توقف.

low :

<meter>

Specifies the range that range that is considered to be a low value.

تحديد اقل قيمة.

max :

<input> , <meter> , <progress>

Specifies the maximum value.

تحديد اعلي قيمة.

maxlenght :

<input> , <textarea>

Specifies the maximum number of characters allowed in an element.

الحد الاقصي.

media :

<a> , <area> , <style> , <link> , <source>

Specifies what media / device the linked document is optimized for.

تحديد نوع الوسائط الذي تم تحسين المستند لة.

method :

<form>

Specifies the HTTP method to use when sending form-data.

لتحديد http لارسال النموذج.

min :

<input> , <meter>

Specifies a minimum value.

لتحديد الحد الادني.

multiple :

<input> , <select>

Specifies that a user can enter more than one value.

للسماح بادخال اكثر من قيمة.

muted :

<video> , <audio>

Specifies that the audio output of the video should be muted.

لالغاء الصوت.

name :

<button> , <fieldset> , <form> , <iframe> , <input> , <map> , <meta> , <object> , <output> , <param> , <select> , <textarea>

Specifies the name of the element.

لتحديد اسم العنصر.

novalidate :

<form>

Specifies that the form should not be validated when submitted.

لالغاء التحقق من صحة النموذج.

onabort :

<audio> , <video> , <img> , <object> , <embed>

Script to be run on abort.

لاحباط تشغيل النص.

onblur :

All visible elements

Script to be run when the element loses focus.

لتفعيلة عند فقد التركيز علية.

oncanplay :

<audio> , <video> , <object> , <embed>

Script to be run when a file is ready to start playing (when it has buffered enough to begin).

ليتم تشغيلة عندما يكون الملف جاهز.

onchange :

All visible elements

Script to be run when the value of the element is changed.

ليتم تشغيلة عند تغير قيمتة.

onclick :

All visible elements

Script to be run when the element is being clicked.

التشغيل عند النقر.

oncopy :

All visible elements

Script to be run when the content of the element is being copied.

التشغيل عند النسخ.

oncut :

All visible elements

Script to be run when the content of the element is being cut

التشغيل عند القص.

ondblclick :

All visible elements

Script to be run when the element is being double-clicked.

لتشغيل البرنامج النصي عند النقر المزدوج للعنصر.

ondrag :

All visible elements

Script to be run when the element is being dragged.

للتشغيل عند السحب.

ondrop :

All visible elements

Script to be run when dragged element is being dropped.

للتشغيل عند ترك سحب العنصر.

ondurationchange :

<audio> , <video>

Script to be run when the length of the media changes.

التشغيل عند تغيير طول الوسائط.

onemptied :

<audio> , <video>

Script to be run when something bad happens and the file is suddenly unavailable (like unexpectedly disconnects).

التشغيل عند تلف الملف او قطع الانترنت.

onended :

<audio> , <video>

Script to be run when the media has reach the end (a useful event for messages like "thanks for listening").

لترك رساله مثل رساله شكر.

onerror :

<audio> , <video> , <style> , <body> , <embed> , <img> , <object> , <script>

Script to be run when an error occurs.

التشغيل عند حدوث خطا.

onfocus :

All visible elements

Script to be run when the element gets focus.

التشغيل عند التركيز علي العنصر.

oninput :

All visible elements

Script to be run when the element gets user input.

التشغيل عند ادخال البيانات.

oninvalid :

All visible elements

Script to be run when the element is invalid.

تفعيلة عندما يتلف العنصر.

onkeyup :

All visible elements

Script to be run when a user releases a key.

لتشغيلة عند الانتهاء من الضغط علي المفتاح.

onkeydown :

All visible elements

Script to be run when a user is pressing a key.

لتشغيلة اثناء الضغط علي مفتاح.

onkeypress :

All visible elements

Script to be run when a user presses a key.

لتشغيلة عندما يضغط المستخدم علي مفتاح.

onload :

<body> , <iframe> , <img> , <input> , <link> , <script> , <style>

Script to be run when the element is finished loading.

لتشغيل البرنامج النصي عند الانتهاء من التحميل.

onloadstart :

<audio> , <video>

Script to be run when media data is loaded.

لتشغيلة قبل بدا التحميل بصفة كلية.

onoffline :

<body>

Script to be run when the browser starts to work offline.

البدء بدون اتصال بالانترنت.

ononline

<body>

Script to be run when the browser starts to work online.

البدء عبر اتصال النت.

onpaste :

All visible element

Script to be run when the user pastes some content in an element.

البرنامج النصي ليتم تشغيله عندما يلصق المستخدم بعض المحتوى في عنصر

onpause :

<video> , <audio>

Script to be run when the media is paused either by the user or programmatically.

تفعيلة عند ايقاف الوسائط يدويا او برمجيا.

onresize :

<body>

Script to be run when the browser window is being resized.

تغيير حجم النافذة.

onscroll :

All visible element

Script to be run when an element's scrollbar is being scrolled.

تشغيل احد العناصر عند تمرير شريط السحب.

onsearch :

<input>

Script to be run when the user writes something in a search field.

البرنامج النصي ليتم تشغيله عندما يكتب المستخدم شيئًا ما في حقل البحث

onselect :

All visible element

Script to be run when the element gets selected.

تفعيلة عند التشغيل.

onstorage :

<body>

Script to be run when a Web Storage area is updated.

تفعيل العنصر عند تحديث الويب.

onsubmit :

<form>

Script to be run when a form is submitted.

تفعيلة عند تقديم النموذج.

ontoggle :

<details>

Script to be run when the user opens or closes the <details> element.

تفعيلة عندما يتم فتحة اوغلقة للعنصر.

onunload :

<body>

Script to be run when a page has unloaded (or the browser window has been closed).

التحميل عند الغاء تحميل الصفحة.

open :

<details>

Specifies that the details should be visible (open) to the user.

تحديد التفاصيل المرئية للمستخدم.

placeholder :

<input> , <textarea>

Specifies a short hint that describes the expected value of the element.

وصف لقيمة المدخل للمستخدم.

pattern :

<input>

Specifies a regular expression that an <input> element's value is checked against.

التحقق من قيمة المدخل.

preload :

<audio> , <video>

Specifies if and how the author thinks the audio/video should be loaded when the page loads.

يحدد ما إذا كان يجب تحميل الصوت / الفيديو عند تحميل الصفحة وكيفية ذلك.

ontoggle :

<details>

Script to be run when the user opens or closes the <details> element

تفعيلة عندما يتم فتح او غلق العنصر.

onunload :

<body>

Script to be run when a page has unloaded (or the browser window has been closed).

التحميل عند الغاء تحميل الصفحة.

rel :

<a> , <area> , <link> , <form>

Specifies the relationship between the current document and the linked document.

لتحديد العلاقة بين الملفات وارتباطها.

required :

<input> , <select> , <textarea>

Specifies that the element must be filled out before submitting the form.

محتوي مطلوب ادخالة اجباري.

reversed :

<ol>

Specifies that the list order should be descending (9,8,7...).

للترتيب التنازلي.

rows :

<textarea>

Specifies the visible number of lines in a text area.

تحديد عدد السطور المحددة.

rowspan :

<td> , <th>

Specifies the number of rows a table cell should span.

تحديد عدد الصفوف المتداخلة.

scope :

<th>

Specifies whether a header cell is a header for a column, row, or group of columns or rows.

تحديد خلية الراس عبارة عن راس العمود او وصف.

selected :

<option>

Specifies that an option should be pre-selected when the page loads.

تحدد أنه يجب تحديد خيار مسبقًا عند تحميل الصفحة.

readonly :

<input> , <textarea>

Specifies that the element is read-only.

للقراءة فقط.

size :

<input> , <select>

Specifies the width, in characters (for <input>) or specifies the number of visible options (for <select>)

تحديد العرض بعدد الاحرف.

span :

<col> , <colgroup>

Specifies the number of columns to span.

تحديد عدد الاعمدة.

step :

<input>

Specifies the legal number intervals for an input field.

تحديد عدد الفقرات.

target :

<a> , <area> , <form> , <base>

Specifies the target for where to open the linked document or where to submit the form.

تحديد الهدف الخاص بمكان الفتح او ارسال النموذج.

title :

Global Attributes

Specifies extra information about an element.

اضافة معلومة للعنصر.

translate :

Global Attributes

Specifies whether the content of an element should be translated or not.

يحدد ما إذا كان يجب ترجمة محتوى العنصر أم لا.

value :

<li> , <button> , <input> , <option> , <meter> , <progress> , <param>

Specifies the value of the element.

تحديد قيمة العنصر.

type :

<a> , <link> , <button> , <embed> , <input> , <menu> , <object> , <script> , <source> , <track>

Specifies the type of element.

تحديد نوع المدخل.

wrap

<textarea>

Specifies how the text in a text area is to be wrapped when submitted in a form.

يحدد كيفية التفاف النص في منطقة النص عند إرساله في نموذج.
